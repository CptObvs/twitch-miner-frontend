## ✨ Features

<table>
<tr>
<td width="50%">

### 🎯 Instance Management
Create and manage multiple miner instances, each with their own Twitch account and custom streamer lists

</td>
<td width="50%">

### 📊 Real-time Monitoring
Live status updates, SSE log streaming, uptime tracking, and process monitoring

</td>
</tr>
<tr>
<td width="50%">

### 👥 Multi-User System
JWT authentication with role-based access control (Admin/User) and isolated instances

</td>
<td width="50%">

### ⚙️ Dynamic Configuration
Edit streamer lists on-the-fly without restarting instances, responsive dashboard

</td>
</tr>
</table>
